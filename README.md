# Alpha Shop 購物車
![AlphaShop_demo](https://user-images.githubusercontent.com/77545400/164825221-44c87729-ee0a-4596-a9af-7fd75ed5cf47.gif)
本專案實作了電商網站內的購物車功能，使用者能夠依循步驟輸入基本資料、選擇運送方式與付款資訊，且右側的購物車隨使用者選擇的選項即時變更包含運費的商品總價。
